# Pokémon-Comparisonatron-3000 - A Python script that fetches real-time Pokémon data from the PokéAPI and visualizes their base stats using a radar chart.

# Features

# Fetches live data directly from the PokéAPI.
# Compares multiple Pokémon simultaneously.Displays stats (HP, Attack, Defense, Special Attack, Special Defense, Speed) on an interactive radar chart.

# Prerequisites

# You need Python installed on your system. This script relies on external libraries to handle API requests and data visualization.

# Installation

# Clone or download this repository to your local machine.
# Open your terminal or command prompt.Install the required dependencies using pip install matplotlib requests numpy

# How to Run

# Run the script from your terminal: python pokemonComparison.py
# Enter the number of Pokémon you want to compare.
# Type the names of the Pokémon one by one (e.g., pikachu, charizard).
# View the generated radar chart visualization.
# Type exit at any prompt to stop adding Pokémon early and generate the chart.
